# INET 4031 - Automated User Creation Script

## Overview
This repository contains two Python scripts used to automate the creation of user accounts on a Linux system for INET 4031 Lab 8 Part 2.

---

## Files

### **create-users.py**
Runs the full automation process:
- Creates user accounts
- Sets passwords
- Assigns groups

Run using:

```bash
sudo ./create-users.py < create-users.input
